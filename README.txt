# Flask Marketplace Project

A simple Flask web application simulating a marketplace where users can register, buy, and sell items.

## Features

- User registration and login (secure with hashed passwords)  
- View, buy, and sell items  
- Budget tracking for users  
- Items assigned to users with database relationship  
- Preloaded sample data (10 users, 10 items)


## Technologies Used

- Python 3.x  
- Flask  
- Flask-Login  
- Flask-Bcrypt  
- Flask-SQLAlchemy  
- SQLite (or any preferred database)
